# MLP–VAE–WGAN Framework

This repository contains the full implementation of the MLP, VAE, and WGAN-based
synthetic data generation and classification framework used in this study.

## Structure
- `src/`: Model implementations and training scripts
- `data/`: Raw, preprocessed, and generated datasets
- `utils/`: Shared utilities and argument parsing

## Reproducibility
All scripts are deterministic given fixed random seeds. Dataset splits used in the
study are provided.

## Requirements
See `requirements.txt`.

## Usage
Scripts can be executed directly from `src/ML_DL`.
